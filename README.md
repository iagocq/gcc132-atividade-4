# Pangrama

Pangrama é uma *frase* que usa **todas** as letras do alfabeto pelo menos uma vez.

Exemplos:
- The quick brown fox jumps over the lazy dog.
- Sphinx of black quartz, judge my vow.
- Todo pajé vulgar faz boquinha sexy com kiwi.
